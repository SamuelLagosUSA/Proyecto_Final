## Descripción

Este proyecto consiste en la implementación del algoritmo ACO (Ant Colony Optimization) para resolver laberintos generados aleatoriamente utilizando Python.

El sistema permite:

- Generar laberintos automáticamente.
- Simular el comportamiento de hormigas artificiales.
- Utilizar feromonas para encontrar rutas eficientes.
- Visualizar gráficamente la evolución del algoritmo.

---

## Tecnologías Utilizadas

- Python
- NumPy
- Matplotlib
- IPython Display

---

## Tema Principal

Aplicación de inteligencia de enjambre para la resolución de problemas de optimización.
